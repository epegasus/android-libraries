# 🧭 Qibla Compass

Accurate, lightweight, and user‑friendly Android app to help users find
the **direction of the Qibla (Kaaba in Mecca)** easily using device
sensors and location services.

## 📍 Minimum Requirements

| Feature | Value |
|---------|-------|
| **Android Minimum SDK** | **API 24 (Android 7.0)** |
| **Language** | Kotlin |
| **UI Toolkit** | Material3 |
| **Sensors** | Accelerometer & Magnetometer |
| **Location** | GPS / Network providers |

------------------------------------------------------------------------

## 🚀 Features

✔ Automatically detects current device location\
✔ Real‑time compass pointing to Qibla direction\
✔ Smooth UI with Material 3 components\
✔ Works worldwide with or without internet\
✔ Minimal permissions & privacy‑friendly

------------------------------------------------------------------------

## 📥 Installation

``` bash
git clone https://github.com/epegasus/Qibla-Compass.git
```

Open in Android Studio → Sync → Run on device.

------------------------------------------------------------------------

## ⚙️ Permissions

-   ACCESS_FINE_LOCATION --- accurate coordinates
-   Sensors --- compass direction

------------------------------------------------------------------------

## 📐 How It Works

1.  Gets device location
2.  Reads orientation sensors
3.  Calculates bearing to Kaaba
4.  Rotates compass UI in real time

------------------------------------------------------------------------

## 🧠 Calibration Tips

-   Move device in figure‑eight motion
-   Avoid metal objects nearby
-   Use outdoors for best accuracy
